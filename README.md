# ITPlace_android01_hw3


За основу берется домашнее задание со второго занятия.
1. Реализовать второе Activity c EditText и Button. В EditText вбивается шестнадцатиричное представление цвета
2. По кнопке должен происходить переход обратно на первое Activity, где введенный цвет применяется в фону обоих TextView.
3. По тапу на ImageView открывать галерею изображений на девайсе и выбранное изображение ставить в ImageView
4. Изображение в Imageview и цвет фона TextView должны переживать поворот экрана
